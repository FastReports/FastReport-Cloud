Для того, чтобы это демо-приложение заработало, нужно настроить ngrok.

1. Скачиваем утилиту с официального сайта - https://dashboard.ngrok.com/get-started/setup

2. Вводим в ngrok команду, чтобы открыть 5001 порт для сети - ​​ngrok http https://localhost:5001 -host-header="localhost:5001"

Теперь нужно настроить отчёт в папке Resources.

1. Закрываем комментарием строку uploadAndExport.Initialize() в конструкторе HomeController'a (32 ст.)

2. Запускаем демо-приложение

3. Пока приложение запущено редактируем строку подключения в отчёте через дизайнер (+ на источниках данных >
 ПКМ по Connection > редактировать > Редактировать...>
 замените адрес подключения к ngrok на тот, что указан в консольном интерфейсе ngrok)

4. Если всё прошло успешно, сохраняем отчёт и закрываем приложение

5. Убираем комментарии с раннее закрытой строки и запускаем приложение
